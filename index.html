Light Unit Converter for CG, HDR, ACES, PBR and Photography

Light Utility Suite is a free online light unit converter and estimation tool for CG, photography, HDR, ACES, PBR, and rendering workflows.
Convert between lumen, candela, lux, nit, EV, exposure, and solid angle.
Values are approximate and based on simplified models.

Simple mode uses recommended defaults: distance = 1 m, sr = 2π, beam angle = 180°, reflectance = 0.8, ISO = 100.

Recommended defaults
Distance: 1 m Solid Angle: 2π sr Beam Angle: 180° Reflectance: 0.8 ISO: 100 Middle Gray: 0.18 (18% gray)

Assumptions

All conversions assume idealized physical models.

  • lm ↔ cd: cd = lm / sr
  • cd ↔ lx: lx = cd / d² (point source, on-axis, normal incidence)
  • lx ↔ nit: nit = lx × reflectance / π (Lambert surface)
  • Stops represent relative exposure (factor of 2 per stop)
  • Exposure multiplier = 2^stops
  • Beam angle ↔ sr assumes uniform intensity in a circular cone (0° < beam angle ≤ 180°, 0 < sr ≤ 2π)
  • EV calculator uses EV100 = log2(N² / t), then adjusts for ISO
  • EV to illuminance uses a standard photographic calibration constant (~2.5 lux at EV0).
  • ACES tool uses scene-linear exposure multiplication around middle gray

Notes

  • This tool was made as a personal utility.
  • All values are approximate and based on simplified models.
  • Do not use for scientific measurement, calibration, or safety-critical applications.
